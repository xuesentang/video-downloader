# 万能视频下载器 - 本地运行指南

## 📋 前置要求

在开始之前，请确保你的电脑已安装以下软件：

1. **Python 3.10+** - 后端运行环境
   - 下载地址：https://www.python.org/downloads/
   - 安装时务必勾选 "Add Python to PATH"

2. **Node.js 18+** - 前端运行环境
   - 下载地址：https://nodejs.org/
   - 建议下载 LTS（长期支持）版本

3. **Git**（可选，用于克隆代码）
   - 下载地址：https://git-scm.com/downloads

## 🚀 快速启动步骤

### 第一步：进入项目目录

```bash
cd C:\Users\28707\Desktop\video-downloader
```

### 第二步：启动后端服务

#### 2.1 创建 Python 虚拟环境（推荐）

```bash
# 进入后端目录
cd backend

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows PowerShell:
venv\Scripts\Activate.ps1
# Windows CMD:
venv\Scripts\activate.bat
# Mac/Linux:
source venv/bin/activate
```

> 💡 **什么是虚拟环境？**
> 虚拟环境可以隔离项目依赖，避免不同项目之间的包版本冲突。就像给每个项目一个独立的"房间"。

#### 2.2 安装 Python 依赖

```bash
# 确保在虚拟环境中（命令行前面有 (venv) 标识）
pip install -r requirements.txt
```

#### 2.3 启动后端服务

```bash
python main.py
```

看到以下输出表示启动成功：
```
🌐 服务启动: http://0.0.0.0:8000
📚 API 文档: http://0.0.0.0:8000/docs
INFO:     Started server process [xxxxx]
INFO:     Uvicorn running on http://0.0.0.0:8000
```

**保持此窗口运行**，不要关闭！

### 第三步：启动前端服务

#### 3.1 打开新的终端窗口

**不要关闭刚才的后端窗口**，重新打开一个新的终端。

#### 3.2 进入前端目录并安装依赖

```bash
cd C:\Users\28707\Desktop\video-downloader\frontend

# 安装前端依赖（只需要第一次运行）
npm install
```

> 💡 **npm install 在做什么？**
> 下载项目需要的 JavaScript 库（如 Vue、Tailwind CSS 等），存储在 node_modules 文件夹中。

#### 3.3 启动前端开发服务器

```bash
npm run dev
```

看到以下输出表示启动成功：
```
VITE v5.4.21  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

**保持此窗口运行**，不要关闭！

### 第四步：访问应用

打开浏览器，访问：**http://localhost:5173/**

你现在应该能看到视频下载器的界面了！

## 🎯 如何使用

### 下载视频的步骤

1. **复制视频链接**
   - 打开 YouTube、Bilibili 等平台
   - 找到想下载的视频
   - 复制浏览器地址栏的 URL

2. **粘贴链接解析**
   - 在应用首页的输入框中粘贴链接
   - 点击"解析视频"按钮
   - 等待解析完成（通常 3-10 秒）

3. **选择清晰度**
   - 解析成功后会显示视频信息
   - 从下拉菜单选择想要的清晰度
   - 通常 1080p 是高清，720p 是标清

4. **下载视频**
   - 点击"下载视频"按钮
   - 浏览器会自动开始下载
   - 文件会保存到你的"下载"文件夹

## 🔧 常见问题

### 1. 后端启动失败："pip 不是内部或外部命令"

**原因**：Python 没有正确添加到系统 PATH

**解决方法**：
1. 重新安装 Python
2. 安装时勾选 "Add Python to PATH"
3. 或者手动添加 Python 安装目录到环境变量

### 2. 前端启动失败："npm 不是内部或外部命令"

**原因**：Node.js 没有正确安装

**解决方法**：
1. 重新下载安装 Node.js
2. 安装完成后重启终端
3. 验证安装：`node -v` 和 `npm -v`

### 3. 虚拟环境激活失败（PowerShell）

**错误信息**：
```
无法加载文件 venv\Scripts\Activate.ps1，因为在此系统上禁止运行脚本
```

**解决方法**：
```powershell
# 以管理员身份运行 PowerShell，执行：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 然后重新激活虚拟环境
venv\Scripts\Activate.ps1
```

### 4. 视频解析失败

**可能原因**：
- 视频链接不正确
- 该平台需要登录才能访问
- 视频有地区限制
- 视频已被删除或设为私密

**解决方法**：
- 检查链接是否正确
- 尝试其他视频链接
- 某些平台（如抖音）可能需要视频是公开状态

- 视频链接不正确
- 该平台需要登录才能访问
- 视频有地区限制

**解决方法**：
- 检查链接是否正确
- 尝试其他视频链接
- 尝试下载其他平台的视频（推荐 YouTube、Bilibili）

### 5. 下载速度慢

**原因**：
- 视频文件较大
- 网络连接不稳定
- 服务器带宽限制

**解决方法**：
- 选择较低清晰度
- 使用更快的网络
- 耐心等待
