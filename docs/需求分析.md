# mian'fei视频下载网站 - 需求分析文档

## 1. 项目背景

用户有从各平台下载保存视频到本地的需求，但很多平台不支持直接下载、或限制较多（清晰度限制、不支持批量下载等）。本项目旨在提供一个统一的、跨平台的万能视频下载工具网站。

## 2. 目标用户

- 需要下载在线视频到本地保存的个人用户
- 需要批量采集视频素材的内容创作者
- 需要在移动端随时下载视频的用户
- 想快速了解长视频内容、提升学习效率的用户

## 3. 核心功能需求

### 3.1 视频解析（P0）

- 用户粘贴视频链接，系统自动解析视频信息
- 展示视频标题、缩略图、时长、上传者、平台来源
- 列出所有可用的格式和清晰度选项

### 3.2 视频下载（P0）

- 用户选择清晰度/格式后，点击下载
- 支持两种下载模式：
  - **直链模式**：解析出视频直链，浏览器直接下载（不占服务器资源）
  - **服务端代理模式**：服务端下载后提供下载链接（兼容有防盗链的平台）
- 自动选择最优下载模式

### 3.3 多平台支持（P0）

- 基于 yt-dlp 开源项目，支持 1800+ 平台
- 包括但不限于：YouTube、Bilibili、抖音、TikTok、Twitter/X、Instagram 等

### 3.4 移动端适配（P1）

- 响应式设计，手机浏览器也能正常使用
- 触屏友好的交互设计

### 3.5 付费功能（预留，当前为免费使用）

- 免费版：基础下载功能，有次数或清晰度限制
- VIP 版：批量下载、4K 画质、字幕下载、无广告、优先队列

### 3.6 AI 视频总结（P1，已实现）

- 用户在视频解析结果页点击「AI 总结」按钮，触发 AI 分析
- **总结摘要**：AI 自动生成视频概述、内容大纲、核心知识要点、一句话总结，使用 @tailwindcss/typography 精美排版
- **字幕文本**：展示带时间戳的完整字幕列表，支持展开/收起，支持下载字幕文件（SRT / VTT / TXT 三种格式）
- **思维导图**：AI 生成结构化内容，前端渲染为可交互的思维导图，支持全屏展示，支持下载高清 PNG（4K+）和 SVG 矢量图
- **AI 问答**：用户可针对视频内容提问，AI 基于字幕上下文回答，支持多轮对话
- 字幕获取策略：优先提取平台自带字幕（人工字幕 > 自动字幕），B 站通过专用 API 获取 CC 字幕
- AI 能力：接入 DeepSeek 大模型，流式输出（SSE），实时展示生成过程

### 3.7 扩展功能（P2，预留）

- 字幕翻译
- 下载历史记录
- 批量下载

## 4. 非功能需求

- **性能**：视频解析响应时间 < 10s
- **可用性**：简洁易用的 UI，新用户无需学习即可使用
- **安全性**：不存储用户数据，临时文件定期清理
- **兼容性**：支持 Chrome、Firefox、Safari、移动端浏览器

## 5. 技术约束

- 以 Python 为主要技术栈
- 核心下载能力依赖 yt-dlp 开源项目
- 不使用数据库，保持轻量
- 前后端分离架构
